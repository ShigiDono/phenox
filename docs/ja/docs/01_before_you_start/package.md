Phenox2 セットには以下のものが含まれています。

- Phenox2 本体
    1. フレーム
    2. プロペラ
    3. モーター
    4. メイン回路
    5. 下面カメラ回路
    6. 正面カメラ回路
    7. シリアル通信回路
    8. プロテクター

- USB ケーブル(A-B)
- バッテリ(11.1V, 180mAh)
- 4GB micro SD カード(環境構築済み)
- USB-WiFi ドングル (WLI-UC, 2)
